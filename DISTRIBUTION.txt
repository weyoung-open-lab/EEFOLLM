EEFOLLM / LLM path planning — distribution package notes
(Companion file for zips produced by scripts/package_release.ps1)

What the package includes
  matlab/ — all MATLAB sources (run_experiment.m is the main entry)
  llm/ — Python weight scripts, prompts, sample I/O; no model weight shards under llm/models/
  maps/ — fixed benchmark grids
  scripts/, requirements-*.txt, README.md, LICENSE

Expected zip size
  A few hundred KB to ~1–2 MB is normal without models, .venv, or results/.

Deliberately excluded (typical)
  Weight files under llm/models/, .venv, .hf_cache, results/, logs/, figures/

Model weights (optional; real Qwen)
  See README.md and llm/models/README.txt.
  Example:
    python llm/download_qwen_model.py --repo Qwen/Qwen2.5-3B-Instruct --out llm/models/Qwen2.5-3B-Instruct-full
  Mock-only: set cfg.use_real_qwen = false in matlab/config/default_config.m

Python
  pip install -r requirements-llm.txt plus a suitable PyTorch wheel.

Third-party licenses
  Comply with Qwen, PyTorch, and other dependencies when publishing.
